# jspdd
基于JSON数据的变更自动产生操作日志，并可自定义日志说明文字。

操作日志精确到 JSON 数据的具体字段.

可记录所有字段的 增/删/改 操作。

## install
    npm install jspdd --save

## importing

    import JSPDD from 'jspdd';
